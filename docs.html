API guia de inicio rapido

Bienvenido a GAPIFS API, aqui puedes aprender a integrar esta aplicacion en tus proyectos, usando la libreria de GIFs y Stickers mas grande del mundo

Inicio rapido

La siguiente guía le guiará a través de la integración mínima viable API GIPHY para una experiencia GIPHY sin problemas en su aplicación. Si tiene alguna pregunta, envíenos un correo electrónico a developers@giphy.com.

Aplicar por una API Key(s)

  1. Crea una clave de API de GIPHY haciendo clic en Crear una aplicación en el panel de desarrolladores (primero tienes que crear una cuenta). Nota: Todas las claves API comienzan como claves beta, que tienen una tasa limitada (42 lecturas por hora y 1000 búsquedas/llamadas API por día).
  2. Crea claves distintas para cada plataforma (iOS, Android, Web) y sección. Si utilizas una integración de GIPHY en diferentes secciones de tu aplicación, GIPHY requiere que cada uso de nuestro servicio esté delimitado por una clave diferente para cada sección por plataforma.

Implementa los Endpoints necesarios

La experiencia GIPHY cobra vida con dos puntos finales esenciales: Tendencias y Búsqueda.

El punto final Trending rellena automáticamente el contenido más relevante y atractivo cada día, mientras que el punto final Search permite a los usuarios buscar en nuestra biblioteca de millones de GIFs y Stickers introduciendo una palabra o frase.
